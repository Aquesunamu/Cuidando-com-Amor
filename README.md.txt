# ONG Cuidar Bem – Sistema Web

Projeto simples para uma ONG de idosos, desenvolvido utilizando:

- HTML
- CSS
- JavaScript
- GitHub
- Vercel

## Funcionalidades
- Página institucional da ONG
- Lista de atividades
- Área de doações
- Formulário de contato com mensagem de confirmação
- Site responsivo

## Como abrir
Basta abrir o arquivo `index.html` em qualquer navegador.
